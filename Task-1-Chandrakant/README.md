# GameDev Mentor Bot

## About

GameDev Mentor Bot is a rule-based chatbot developed in Python for my Decode Labs AI Internship.

I created this project because I want to become a game developer. During my learning journey, I built simple 2D and 3D games and learned from many beginner mistakes. This chatbot helps beginners by answering common game development questions.

## Features

* Career guidance
* Learning roadmap
* Programming tools
* AI in game development
* Beginner project ideas
* Common mistakes

## Technologies

* Python
* Dictionary
* Loops
* Conditional Statements

## Commands

`hello` `career` `learning` `tools` `games` `ai` `mistakes` `help` `bye`

## Run

```bash
python chatbot.py
```
