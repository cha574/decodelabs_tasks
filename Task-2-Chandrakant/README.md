# Iris Flower Classification using KNN

This project is a part of my AI Internship at DecodeLabs.

The goal of this project is to build a basic machine learning classification model using the Iris dataset. The model is trained using the K-Nearest Neighbors (KNN) algorithm and can predict the species of a flower based on its measurements.

## Features

- Load the Iris dataset
- Convert the dataset into a Pandas DataFrame
- Display dataset information and statistical summary
- Split the dataset into training and testing data
- Scale features using StandardScaler
- Train a KNN classification model
- Evaluate the model using Accuracy, Confusion Matrix, and Classification Report
- Visualize the dataset using Matplotlib
- Predict the species of a new flower using user input

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning Workflow

1. Load the dataset
2. Prepare features and labels
3. Split the dataset into training and testing sets
4. Scale the features
5. Train the KNN model
6. Make predictions
7. Evaluate the model
8. Predict a new flower from user input

## Dataset

The project uses the **Iris Dataset** available in the Scikit-learn library.

Dataset Details:
- Total Samples: 150
- Features: 4
- Classes: 3
  - Setosa
  - Versicolor
  - Virginica
